Sobre mí:

Me encanta aprender. Pudiera ser cómo construir un nuevo sitio, la historia de el libro nuevo, cómo trabajar un grupo muscular de la manera más efectiva, o cualquier otra cosa. Esa es mi pasión. Con un amplio camino abierto frente a mí, este es el camino que elijo seguir. El aprendizaje es lo más importante que cualquiera de nosotros puede hacer. Así que te animo, ¡únete a mi viaje y comienza el tuyo propio!

Mis proyectos:

Estos serán una colección de los proyectos completados bajo Team Treehouse en la certificación Técnica en Desarrollo Web Front End. El proyecto final es el que está viendo ahora.

Mi página de perfil personal

El primer proyecto. Su objetivo: enseñar los conceptos básicos de html estructuración y maquetación.

Mi página de perfil personal

Aunque básico, este proyecto me permitió conocer los conceptos básicos del estilo html y css. Los desafíos de este proyecto me permitieron trabajar en desafíos posteriores más difíciles y fue el elemento fundumental de todo mi conocimiento actual.

Un formulario de registro en línea

El segundo proyecto. Su objetivo: enseñar estilos css, estructuración html, formularios html.

Un formulario de registro en línea

Durante las primeras etapas de este curso, este fue sin duda uno de mis proyectos favoritos, ya que en el controlo todo el estilismo. ¡Estoy muy orgulloso de la forma en que todo se unió para producir este formulario html!

Aplicación de programa de juegos

El sexto proyecto. Su objetivo: conceptos básicos de JavaScript y manipulación de DOM.

Aplicación de programa de juegos

Este fue uno de los primeros proyectos en los que me divertí mucho haciéndolo. Como un juego de adivinanzas de palabras cortas te permite ver debajo del "capó" de un sitio web y manipularlo exactamente como te gustaría. ¡Uno de mis favoritos!

Panel de control de la aplicación web

El séptimo proyecto. Su objetivo: incorporar todas las habilidades anteriores en la estructuración de html, estilos css y sass, manipulación DOM de JavaScript y uso de una biblioteca externa.

Panel de control de la aplicación web

Apodado la "bestia", fue uno de los proyectos más difíciles y que más tiempo consumió de todo el curso. Incorporó todas las habilidades utilizadas en los proyectos anteriores, además de algunas más, para crear una Webbapp interactiva. El proyecto incluso incluye la validación de formularios en sus elementos de formulario. Aunque no necesariamente es uno de mis proyectos favoritos ¡Estoy muy orgulloso del resultado final!

API para crear un panel de empleados

El penúltimo proyecto. Su objetivo: usar una API para manipular el DOM, agregar contenido html utilizando JavaScript y diseño de cuadrícula.

API para crear un panel de empleados

Debo decir que este fue mi proyecto favorito de todos. Uso de contenido de un sitio web completamente diferente me abrió las posibilidades de todo lo que se puede hacer con el desarrollo de software y la codificación en su conjunto. Estoy increíblemente emocionado para continuar mi camino a través del desarrollo de software y la programación informática.
