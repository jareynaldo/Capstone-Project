Sobre mí:

Me encanta aprender. Pudiera ser cómo construir un nuevo sitio, la historia de el libro nuevo, cómo trabajar un grupo muscular de la manera más efectiva, o cualquier otra cosa. Esa es mi pasión. Con un amplio camino abierto frente a mí, este es el camino que elijo seguir. El aprendizaje es lo más importante que cualquiera de nosotros puede hacer. Así que te anímo, ¡únete a mi viaje y comienza el tuyo propio!

Mis proyectos:

Esto es un recorrido cronológico de mis proyectos. En cima están los proyectos que aprendi al principio. Empezando con una estructura HTML simple, mientras que más abajo están los más recientes y complicados.

Mi página de perfil personal

Aunque básico, este proyecto me permitió conocer los conceptos básicos de HTML y CSS. Este fue el primer paso en mi viaje.

Un formulario de registro en línea

Después de haber aprendido cómo funciona la estructuración HTML, me sumergí en el mundo del estilo con CSS. Aquí aprendí cómo entrelazar HTML funcional con CSS para decorar.

Aplicación de programa de juegos

Después de dominar HTML y CSS, llegó el momento de agregar interactividad a mis proyectos con JavaScript. Aquí utilicé JS para cambiar constantemente los elementos del sitio y hacerlo dinámico.

Panel de control de la aplicación web

Para este proyecto, simplemente me dieron un conjunto de maquetas de cómo debería verse el sitio y lo implementé. Se necesitó estructuración HTML, diseño SCSS, interactividad JS y gráficos interactivos, manipulación SVG y se aseguró de que todo fuera compatible con dispositivos móviles.

API para crear un panel de empleados

Durante este proyecto, adquirí la habilidad de interactuar con una API de terceros a través de solicitudes personalizadas para recuperar datos JSON. El contenido del sitio web se genera dinámicamente con cada carga de página, lo que muestra la automatización en acción. Además, amplié mis conocimientos implementando una barra de búsqueda y mejorando la interactividad dentro del perfil de cada empleado.

Proyecto Átomo

Este proyecto fue parte de una presentación más amplia sobre el átomo. Una de las formas más sencillas de aprender es a través de la interactividad, en la que este sitio destaca.

Energía nuclear

Siguiendo el proyecto atómico, la energía nuclear sirve como una forma de desmitificar la energía nuclear para la persona promedio. Es mucho más agradable visualmente en comparación con el proyecto anterior y hace un trabajo increíble al explicar.

La Nueva Esperanza

Todos los proyectos anteriores sucedieron antes del último año. Como resultado, decidí practicar mis habilidades de desarrollo web en un escenario de la vida real y me ofrecí como voluntario en mi iglesia local para crear un sitio web. Aquí creé un sitio web fácil de usar, colaborando con el personal para asegurarme de que cumpliera con sus requisitos.
